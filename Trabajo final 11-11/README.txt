############# Descripción #############
Se trata de un programa el cual fue diseñado para trabajar con un archivo .CSV, en el mismo
se almacenan países con sus nombres, poblaciones, territorios y continentes.
El programa permite la modificación del archivo para agregar o cambiar datos, como también
brinda estadísticas de los países dentro del archivo .CSV .
############# Uso #############
El programa se debe ejecutar en un editor de código (preferiblemente Visual Studio Code).
El programa se ejecuta por terminal, el usuario solo debe elegir las opciones que quiera.
NO hace falta contar con un archivo .CSV para ejecutar el programa, el mismo lo creara si
no existe.
############# Inputs y outputs #############
El programa podrá pedir al usuario ingresar países y sus respectivos datos, a si mismo 
brindara información como promedios o estadísticas de los países existentes.
